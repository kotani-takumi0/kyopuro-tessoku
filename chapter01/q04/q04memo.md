## ビットシフト演算子
ビットシフト演算子は「数字の2進数の桁を横に動かす」操作
例
6を二進数で書くと
0000 0110
ここで、左へ一桁動かすと
0000 1100
これは12になる

つまり
x << 1 は左へ桁を一つずらすので、x*2
x << 2 は左へ桁を二つずらすので、x*4

x >> 1 は右へ桁を一つずらすので、x*1/2
x >> 2 は右へ桁を二つずらすので、x*1/4

注意
x >> 1 と 1 << x は全くの別物
- x >> 1：xを右へ一ビットずらす操作（xを2で割る）
- 1 << x：2^xを作る操作（今回の場合）
### 今回の問題での使い方
1 << xを使用して、2^xを作っている
この式では1をxビットずらすということである
例
0000 0001 (x = 1)
↓　x = 3
0000 1000 (x = 2^3 = 8)
